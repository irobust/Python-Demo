## Useful function for inspection
* help()
* id()
* bool()
* dir()
* type()
* [int.__class__]
* [basic.__annotations__]
* import inspect
* sig = inspect.signature(funcName)
* sig.parameters['parameterName']
* issubclass(a, b) # a is subclass of b
* isinstance(a, b) # a is instance of basic
* globals()
* locals()
